# Краулер для генерации и экспорта данных в CSV
поддерживает форматы docx, xls, xlsx, pdf, а также их вложенностей в архивы zip
doc, rar, 7z реализовать не получилось, работало через раз, поэтому пришлось убрать вообще.

Этот проект представляет собой краулер, который генерирует данные, сохраняет их в базу данных и позволяет экспортировать данные в формате CSV. Проект состоит из следующих файлов:

- `config.py` — файл конфигурации.
- `generate.py` — генератор данных.
- `script.py` — основной скрипт для работы с базой данных. Записывает инфу о доках в папке документс. 
- `export.py` — выгружает csv таблицы из бд

## Установка

1. Убедитесь, что у вас установлен Python 3.x.
2. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
Использование
Шаг 1: Генерация данных
Сначала запустите генератор данных, чтобы создать необходимые записи для базы данных. Для этого выполните следующую команду перейдя в ../../краулер/crowler:

python generate.py
Шаг 2: Запуск скрипта
После генерации данных выполните основной скрипт, который будет обрабатывать данные и сохранять их в базу данных:

python script.py
Шаг 3: Экспорт данных в CSV
После того как данные будут сохранены в базе данных, вы можете экспортировать их в файл CSV. Для этого выполните следующую команду:

python export.py
Проверка работы
Вы можете проверить, что данные были успешно сохранены и экспортированы, открыв сгенерированный CSV-файл. Он должен содержать все данные, которые были сгенерированы и сохранены в базе данных.

Примечания
Убедитесь, что в файле config.py и в export 
if __name__ == "__main__":
    db_config = {} 
    правильно указаны параметры подключения к вашей базе данных.
